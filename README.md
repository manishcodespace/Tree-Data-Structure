# Tree Data Structure

This project contains TypeScript implementations of Tree Data Structure algorithms.

## Features

- **Binary Tree Implementation**: Basic structure definition (`TreeNode`) and class (`BinaryTree`).
- **Inorder Traversal**: Left -> Root -> Right traversal printing the result in a single line.

## Getting Started

### Prerequisites

- Node.js installed
- TypeScript installed globally or accessible via npx

### Compilation and Execution

1. **Compile the TypeScript code:**

   ```bash
   tsc
   ```

2. **Run the compiled JavaScript:**

   ```bash
   node dist/Inorder.js
   ```

   **Output:**
   ```
   4 2 5 1 6 3 7
   ```

### Alternative (using ts-node)

You can also run the TypeScript file directly using `ts-node`:

```bash
npx ts-node src/Inorder.ts
```
